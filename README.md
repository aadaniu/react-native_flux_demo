##Setup
1. Follow the "Getting Started" guide from [Facebook's React Native GitHub](https://facebook.github.io/react-native/docs/getting-started.html#content)<br />
  ```sh
  $ npm install
  ``` 
<br /><br />
2. Open the .xcodeproj file and hit Run in Xcode.
